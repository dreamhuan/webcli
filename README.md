dreamhuan <fu_kaiqi@qq.com>  
github: [webcli](https://github.com/dreamhuan/webcli)
  
### 安装
npm i -g webcli

### 删除
npm uninstall -g webcli

### 使用
1. webcli -h ：显示说明
1. webcli new \<name\> ：在当前目录下新建name名字的project并建立初始项目文件（会新建name文件夹）
